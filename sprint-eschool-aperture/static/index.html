About Us

📝 Aperture Notes is an interactive notepad designed to transform traditional note-taking into a more engaging and effective learning experience. With Aperture Notes, users can write notes, embed videos, and add images to better understand topics and overcome the monotony of studying. This innovative approach makes learning more dynamic, personalized, and efficient.

🚀 How to Use Aperture Notepad

  • 📚 Log in:Click the proceed button and login or register, if you havent yet.
  • ✍️ Add a lesson: When already in the lessons menu you acn easily add a lesson clicking the "+" sign and giving it a name in the prompt above
  • 🖼️ Start Writing: When the lesson is created you will be able to start taking notes in the editable area.
  • 🎥 Embed Media: If you want to add an image to your notes you can upload an image using the upload image button.
  • 📂 Organize Efficiently: If you want to find a specific word you can try highlighting it using the find button in the top righ. (In the far left side)
  • 🪧 Active Recall: If you want memorize and learn the lesson faster you can create a small 20 question quiz using the AI tester button in top right. (In the middle)
  • 💾 Save & Review: After you've finished writing you can easily save using the button in the top right corner resembling a save icon.(In the far right side)
